# Lesson 1: Binary - The Language of Computers

## Big Idea
Computers understand only binary (0 and 1). Every type of data—text, images, audio, and video—is ultimately represented using binary.

---

## Why?
Electronic circuits naturally have two stable states:
- OFF → 0
- ON → 1

Using two states makes computers fast and reliable.

---

## Important Concepts

### Decimal (Base 10)
Uses digits 0–9.
Place values are powers of 10.

Example:
347 = 3×10² + 4×10¹ + 7×10⁰

---

### Binary (Base 2)
Uses digits 0 and 1.
Place values are powers of 2.

Example:
101₂ = 1×2² + 0×2¹ + 1×2⁰ = 5₁₀

---

## Important Formula

Number of combinations using n bits:

2ⁿ

Largest unsigned value:

2ⁿ − 1

---

## AI Connection

Every image, neural network weight, word embedding, and model parameter is stored as binary inside computer memory.

---

## Things I Learned

- Binary is used because hardware is reliable with two states.
- More bits mean more possible values.
- 8 bits = 256 possible values.
- RGB images use 24 bits per pixel.
- Images are stored as numbers.
